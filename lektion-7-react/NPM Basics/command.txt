NPM
---------------------------------------------------------------------------------------------------
npmjs.com

npm init                                    skapar package.json
npm install                                 installerar alla dependencies i ett projekt
npm install <package name>                  installerar npm paketet lokalt i projektet
npm install <package name> -g               installerar npm paketet globalt
npm install <package name> --save-dev       installerar npm paketet endast för utveckling  


ex.
npm init
npm install create-react-app -g
npm install bootstrap
